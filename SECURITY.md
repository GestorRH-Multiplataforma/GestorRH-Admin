# Security Policy

## Supported Versions

Only the latest stable release receives security updates.
Older versions are not actively maintained.

## Reporting a Vulnerability

If you discover a security vulnerability in GestorRH-Admin, please report it
privately through GitHub's security advisory feature instead of opening a
public issue.

You can expect an initial response within 72 hours. If the vulnerability is
confirmed, a patch will be prioritized for the next release.
